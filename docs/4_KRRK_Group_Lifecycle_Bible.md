# Group 철학
*Group은 KRRK의 모든 것이 쌓이는 영구 공간이다. 친구들이 모이든 안 모이든, 우리 그룹은 살아있다.*

그룹은 삭제되지 않는다. 기록은 사라지지 않는다.
3년치 독박왕 연대기, 역대 시즌 기록, 업적 — 전부 이 공간에 쌓인다.
오래 쓸수록 다른 앱으로 이동할 이유가 없어진다. 그룹이 해자다.

# Group 기본 스펙

# Group 라이프사이클
## STEP 1 — 그룹 생성
누구나 그룹을 만들 수 있다. 만든 사람이 방장이 아니다. 그냥 첫 번째 멤버다.

**방식 A — 링크 하나 공유**
1. 그룹 이름 입력 → 그룹 생성 완료.
2. 초대링크 자동 생성. 유효기간 선택 (24시간 / 7일).
3. 단톡방에 링크 하나 공유.
4. 각자 링크 클릭 → 닉네임 입력 → 게스트 입장.

인원이 확정되지 않았을 때. 누구나 들어올 수 있어야 할 때.

**방식 B — 개인 링크 생성**
1. 그룹 이름 입력 → 참여자 이름 전원 미리 등록.
2. 참여자별 개인 링크 자동 생성.

철수  →  https://krrk.io/aaa1
영희  →  https://krrk.io/aaa2
순자  →  https://krrk.io/aaa3
1. 전체 링크 목록을 단톡방에 한 번에 공유.
2. 각자 자기 이름 링크 클릭 → '철수가 맞습니까?' 확인 → 입장.
3. 잘못된 링크 클릭 시 '이 링크는 영희용입니다. 철수 링크로 다시 입장해주세요' 안내.

인원이 확정됐을 때. 닉네임 입력 단계 없음. 가장 빠른 입장 방식.

*그룹 생성은 30초 이내. 두 가지 방식 중 상황에 맞게 선택.*

## STEP 2 — 멤버 입장
초대링크를 받은 사람은 브라우저에서 바로 입장. 설치 없음.

**입장 플로우**
1. 링크 클릭 → 브라우저 열림.
2. 닉네임 입력 → 게스트로 바로 입장.
3. 그룹 홈 진입. 활성 챌린지 확인 가능.
4. 게임 후 개인 토큰 링크 발급. 카카오 나한테 보내기로 저장.

**초대링크 만료 시**
- 기존 멤버는 개인 토큰 링크로 재입장 가능. 영향 없음.
- 새 멤버 초대가 필요하면 누구나 새 초대링크 생성 가능.

## STEP 3 — 활성 상태
챌린지가 진행 중이거나 최근 3개월 내 게임이 있으면 활성 상태다.

**활성 그룹에서 할 수 있는 것**
- 챌린지 제안 및 참여.
- 시즌 기록 및 히든 랭킹 확인.
- 그룹 타임라인 조회 (역대 시즌 결과, 독박왕 연대기).
- 새 멤버 초대링크 생성.
- 그룹 이름 변경 (누구나 가능).

## STEP 4 — 휴면 전환
마지막 게임으로부터 3개월이 지나면 자동으로 휴면 모드로 전환된다.

**휴면 전환 시**
- 그룹 목록에서 사라짐. 별도 휴면 탭으로 이동.
- 진행 중인 챌린지가 있으면 휴면 전환 전에 알림 발송.
- 기록 전부 유지. 조회 가능.

**휴면 중에 할 수 있는 것**
- 기록 조회만 가능.
- 챌린지 제안 불가.

*아무도 총대 안 메도 된다. 자동으로 조용히 내려간다.*

## STEP 5 — 휴면 복구
휴면 그룹은 언제든 살릴 수 있다.

**복구 플로우**
1. 휴면 탭에서 그룹 선택.
2. '그룹 다시 시작하기' 탭.
3. 즉시 활성 상태로 복구. 기록 전부 유지.
4. 새 챌린지 제안 가능.

오랜만에 만나는 친구들이랑 '야 우리 KRRK 다시 하자' 하면 — 역대 기록 보면서 다시 시작할 수 있다.

# Group State Machine

| ACTIVE | 3개월 내 게임 있음. 챌린지 가능. 그룹 목록 노출. |
| --- | --- |

| DORMANT | 3개월 이상 게임 없음. 자동 휴면. 목록에서 숨김. 기록 유지. |
| --- | --- |

| RESTORED | 휴면 복구. 즉시 ACTIVE로 전환. |
| --- | --- |

# 그룹 홈 구성
그룹에 입장하면 보이는 화면.

| 섹션 | 내용 |
| --- | --- |
| 활성 챌린지 | 현재 진행 중인 챌린지 목록. 참여 버튼. |
| 내 상태 | 현재 챌린지에서 나의 히든 랭킹 상태 (안전권/위험권/매우 위험). |
| 챌린지 제안 | 누구나 새 챌린지 제안 가능. |
| 그룹 타임라인 | 역대 시즌 결과, 독박왕 연대기, 업적. |
| 멤버 | 그룹 멤버 목록. 각자 누적 기록 확인. |

# 시나리오 예시
## 시나리오 A — 첫 만남, 그룹 생성
민수, 지훈, 현우, 태양 — 오랜만에 술자리.

1. 현우가 KRRK 접속. 그룹 이름 '대학친구' 입력. 30초.
2. 초대링크 생성. 유효기간 24시간 선택.
3. 단톡방에 링크 공유.
4. 민수, 지훈, 태양 링크 클릭. 닉네임 입력. 바로 입장.
5. 현우가 챌린지 제안. 게임 시작.
6. 게임 후 전원 개인 토큰 링크 카카오 나한테 보내기로 저장.

## 시나리오 B — 3개월 후 재회
3개월 만에 다시 만나는 대학친구들.

1. 그룹이 자동 휴면 상태.
2. 민수가 휴면 탭에서 '대학친구' 그룹 발견.
3. '그룹 다시 시작하기' 탭. 즉시 활성화.
4. 역대 독박왕 기록 확인. '지훈이 작년에 3번 긁혔네 ㅋㅋ'
5. 새 챌린지 제안. 다시 게임 시작.

## 시나리오 C — 새 멤버 추가
기존 그룹에 새 친구를 데려오는 경우.

1. 지훈이 새 초대링크 생성. 유효기간 7일 선택.
2. 링크를 새 친구 승현에게 전달.
3. 승현 입장. 기존 그룹 기록 전부 볼 수 있음.
4. 다음 챌린지부터 자동 참여.

# Design Principles

- **그룹 생성은 30초 이내.**
- **방장 없음. 전원 평등.**
- **삭제 없음. 기록은 영구 보존.**
- **휴면은 자동. 아무도 총대 안 메도 된다.**
- **복구는 언제든. 역사는 그대로.**
- **초대링크 만료는 보안. 기존 멤버 재입장에 영향 없음.**
