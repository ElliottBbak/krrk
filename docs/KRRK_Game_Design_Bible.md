# Game 철학
*게임은 수단이다. 친구들 사이의 결정을 드라마로 만드는 수단.*

**설계 원칙**
- 10초 안에 이해. 30초 안에 종료.
- 실력보다 운. MVP는 전원 랜덤 기반 (타이머 게임 제외).
- 관전자도 재밌어야 한다.
- 결과가 드라마틱해야 한다.
- 캡처해서 단톡방에 뿌리고 싶어지는 결과 화면.
- 구슬에 멤버 닉네임/아바타 표시. 내 친구가 지는 게 재밌다.

# MVP 게임 3종

| 게임 | 핵심 감정 | 방식 | 꼴찌 정하기 | 1등 정하기 |
| --- | --- | --- | --- | --- |
| 🔵 구슬 레이스 | 역전, 추월, 탈락 | 랜덤 낙하 | ✓ | ✓ |
| 💣 폭탄 게임 | 제발 나 아니길 | 랜덤 패스 | ✓ | ✗ |
| ⏱ 타이머 게임 | 손 떨림, 심리전 | 0.01초 스톱 × 2 | ✓ | ✓ |

폭탄 게임은 꼴찌 정하기 전용. 1등 베네핏 챌린지에서는 선택 불가.

🔵 **구슬 레이스**  — 운명은 핀에 맡긴다

**기본 스펙**

| 항목 | 내용 |
| --- | --- |
| 플레이 시간 | 20~30초 |
| 인원 | 2명 이상 |
| 방식 | 랜덤 낙하. 실력 없음. |
| 레퍼런스 | lazygyu.github.io 구슬 레이스 |
| 꼴찌 결정 | 가장 늦게 도착한 구슬의 주인 |

**게임 방식**
1. 멤버 수만큼 구슬 생성. 각 구슬에 멤버 닉네임 + 아바타 표시.
2. 화면 상단에서 구슬들이 동시 출발.
3. 핀, 레일, 분기점을 거치며 랜덤으로 방향 변경.
4. 먼저 도착한 순서대로 순위 결정.
5. 마지막 도착 = 꼴찌.

**핵심 연출**
- 왼쪽 미니맵으로 전체 경로 실시간 확인 가능.
- 분기점마다 역전 가능 → 관전자 반응 유도.
- 내 친구 구슬이 선두에서 꼴찌로 역전당하는 순간이 하이라이트.
- 결과 화면: 순위표 + 꼴찌 강조 연출 + 단톡방 공유 버튼.

**GUI 방향**
- 레퍼런스 메커니즘 유지. GUI는 KRRK 감성으로 리디자인.
- 네온/다크 배경 유지. 구슬 디자인은 과일 대신 멤버 아바타.
- B급 밈 감성 효과음 및 애니메이션.

💣 **폭탄 게임**  — 제발 나 아니길

**기본 스펙**

| 항목 | 내용 |
| --- | --- |
| 플레이 시간 | 10~30초 |
| 인원 | 2명 이상 |
| 방식 | 랜덤 패스. 실력 없음. |
| 꼴찌 결정 | 폭탄이 터질 때 보유하고 있는 사람 |

**게임 방식**
1. 폭탄 하나가 멤버들 사이를 랜덤으로 이동.
2. 타이머는 보이지 않음. 언제 터질지 모름.
3. 폭탄 보유 중인 멤버 화면에 긴장감 연출.
4. 5초 남으면 '터지기 직전' 신호 발동.
5. 폭탄이 터지면 그 순간 보유자가 꼴찌.

**핵심 연출**
- 평소: 폭탄이 화면에서 가만히 있음.
- 5초 전: 폭탄이 미친듯이 흔들리면서 연기 뿜기 시작.
- 보유자 화면: 더 과장된 흔들림 + 화면 테두리 빨갛게 깜빡임.
- 관전자 화면: 현재 누가 들고 있는지 표시.
- 폭발 순간: 드라마틱한 폭발 애니메이션 + 꼴찌 확정 연출.
- 결과 화면: '긁혔다 💣' + 꼴찌 닉네임 강조 + 공유 버튼.

**타이머 설계**
- 총 시간: 10~30초 랜덤. 참여자 포함 아무도 모름.
- 5초 전 신호: 폭탄 흔들림 + 연기 연출만. 카운트다운 숫자 없음.

*타이머가 안 보이기 때문에 5초 전 신호가 나왔을 때 '내가 들고 있으면 멘탈 붕괴'가 연출된다.*

⏱ **타이머 게임**  — 손이 떨리면 진다

**기본 스펙**

| 항목 | 내용 |
| --- | --- |
| 플레이 시간 | 30~60초 |
| 인원 | 2명 이상 |
| 방식 | 0.01초 단위 타이머 두 번 스톱. 심리전 포함. |
| 꼴찌 결정 | 마지막 자리 수 곱하기 값이 가장 높은 사람 |
| 동점 처리 | 재대결 |

**게임 방식**
1. 순서 정하기 버튼 → 랜덤으로 플레이 순서 확정. 모든 화면에 순서 표시.
2. 1번 차례 — 해당 멤버가 타이머 스타트. 모든 화면에 실시간으로 타이머 표시.
3. 1번이 원하는 타이밍에 첫 번째 스톱 → 소수점 마지막 자리 수 확정. 전체 공개.
4. 타이머 다시 시작. 두 번째 스톱 → 두 번째 자리 수 확정. 전체 공개.
5. 두 자리 수 곱하기 → 결과 확정. 전체 화면에 표시.
6. 2번, 3번... 순서대로 반복. 앞 사람 결과를 보면서 해야 하는 압박.
7. 전원 완료 후 최고값이 꼴찌. 동점이면 재대결.

**계산 예시**

| 멤버 | 1차 스톱 | 2차 스톱 | 결과 (곱하기) |
| --- | --- | --- | --- |
| 민수 | 3.47 → 7 | 8.52 → 2 | 7 × 2 = 14 |
| 지훈 | 5.23 → 3 | 6.89 → 9 | 3 × 9 = 27 |
| 현우 💣 | 7.96 → 6 | 9.18 → 8 | 6 × 8 = 48 ← 꼴찌 |

**핵심 연출**
- 순서 공개 순간 — '내가 마지막이다' vs '내가 1번이다' 반응 차이.
- 내 차례 아닌데 타이머가 모든 화면에 보임 → 관전자도 같이 쫄림.
- 앞 사람 결과 보이는 상태에서 내 차례 → '나 지금 X 이하로 나와야 해' 압박.
- 1차 스톱 후 큰 수가 나오면 → 2차에 작게 멈춰야 한다는 압박 + 손 떨림.
- 마지막 순서 멤버가 제일 쫄림. 이미 최고값 알고 있으니까.
- 꼴찌 확정 순간 '긁혔다 ⏱' 연출 + 전체 결과표 공개.
- 동점 재대결 시 긴장감 두 배.

**심리전 포인트**
*마지막 순서가 제일 쫄린다. 앞 사람들 결과 다 보이는데 내가 지금 얼마 이하로 나와야 살지 알면서 손이 떨리는 거다.*

# 결과 화면 공통 원칙

모든 게임의 결과 화면은 동일한 원칙을 따른다.

| 요소 | 내용 |
| --- | --- |
| 꼴찌 강조 | 꼴찌 닉네임 크게. '긁혔다' 텍스트. 과장된 애니메이션. |
| 전체 순위 | 1위부터 꼴찌까지 순위표 표시. |
| 챌린지 반영 | 이 결과가 현재 챌린지 시즌에 자동 반영됨 안내. |
| 공유 버튼 | 결과 카드 이미지로 캡처. 단톡방 공유 가능. |
| 다시 하기 | 같은 설정으로 바로 한 판 더. |
