배포 방식
MVP는 CI/CD 없이 수동 배포. 코드 push → EC2 접속 → git pull → 빌드 → pm2 재시작.
프론트 배포
cd apps/web && pnpm build
cp -r dist/ /var/www/krrk/web/dist/
빌드 결과물을 nginx 서빙 경로에 복사. nginx 재시작 불필요.
백엔드 배포
cd apps/api && pnpm build
pm2 restart krrk-api
pm2가 프로세스 재시작. 다운타임 최소화.
