프론트엔드 아키텍처
Zustand — 실시간 상태
gameStore현재 게임 세션 상태. 소켓 이벤트 수신 시 업데이트. 마블 위치, 폭탄 보유자, 타이머 현황.
authStoreuserId, accessToken, personalToken. localStorage에 persist.
socketStore소켓 연결 상태. 이벤트 핸들러 등록/해제.
TanStack Query — 서버 데이터
useGroup그룹 홈 데이터. 멤버 목록, 활성 챌린지. 자동 캐싱.
useChallenges챌린지 목록 + 상태. 폴링 또는 소켓 이벤트 시 invalidate.
useMyStatus히든 랭킹 내 상태 (SAFE/DANGER/CRITICAL).

JWT 인증 전략
Refresh Token 없음. 게스트 유저 특성상 토큰 분실 시 재식별 불가이므로 복잡도 추가 불필요. 소셜 로그인(Phase 4) 때 Refresh Token 추가.
유저 타입accessToken 만료재발급 방법
GUEST30일개인 토큰 링크 → /auth/token → 새 accessToken
TOKEN (개인링크)1년개인 링크 자체가 영구 식별자
SOCIAL7일Phase 4 때 Refresh Token 추가 예정
